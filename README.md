# 介護の輪

## サイト概要
### サイトテーマ
- 介護に携わる家族や介護士が情報交換や相談、ストレス解消などができるコミュニティサイト

### テーマを選んだ理由
- 前職で介護の仕事をしており、介護をしている人の悩みの相談や情報交換、ストレスの解消方法等を共有できるサイトがあれば便利だと考え、このテーマにしました。

### ターゲットユーザ
- 介護経験のある人

### 主な利用シーン
- 介護の仕方で悩んだとき
- 相談に乗ってほしいとき
- 初めて介護を経験するとき等
